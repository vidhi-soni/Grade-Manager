# Grade-Manager

To run the program: 

    1. First compile driver.java in terminal 
    2. Then run driver.java
    3. The program will ask for user input:
       -'a' for average grades
       -'d' to display grades 
       -'h' to show highest grade
       -'f' to fix all grades
       -'r' to assign random grades
       -'q' to quit
